# Vibe player
The music player with the function of matching music to your mood 

## Features
- Listen to your music
- Upload your own music
- Delete music
- Speed up and slow down music

## Technologies
- Electron (includes JS, HTML, CSS)

## Installation
1) Go to the latest release
2) Click on the last version
3) Download the installer file for your OS (Windows -> .exe, Linux -> .deb, MacOS -> .dmg)
3) Open installer
